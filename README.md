# MyFxxkingNordicBot

A lightweight terminal AI coding assistant, forked from [MiniCode](https://github.com/LiuMengxuan04/MiniCode).

## Quick Start

```bash
npm install
npm run install-local
minicode
```

## Development

```bash
npm run dev                              # dev mode
MINI_CODE_MODEL_MODE=mock npm run dev    # offline mode
npm run check                            # type check
```
